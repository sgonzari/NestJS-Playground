# Pasos a seguir
## Descargar cli de NestJS de manera global
```
npm i -g @nestjs/cli
```

## Creación del proyecto
```
nest new project-name
```
_Eliminar carpeta .git para poder subir al repositorio_

## Creación de un módulo
```
nest g module module-name
```
